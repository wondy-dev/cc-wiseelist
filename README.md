# cc · wiseelist

초보는 벗어난, 클로드 코드 좀 쓰는 법 — **심화편**.

토큰은 아끼고 품질은 올리는, 클로드 코드를 1인 회사처럼 운영하는 법을 7개 장으로 정리한 가이드입니다.

🔒 본문은 비밀번호로 보호되어 있습니다. 접근 권한은 공유받은 분에게 문의하세요.

다루는 내용

1. 빠르고 완성도 높게 계획 짜기 (플랜 모드 · 필수 질문 5가지)
2. 에이전트를 효율적으로 굴리기 (경로 전달 · JSON 핸드오프 · 병렬)
3. 직급별 모델 나누기 (하네스 구성)
4. 거버넌스 효율적으로 세우기 (CLAUDE.md · 규칙 계층 · 조건부 로딩)
5. 쌓이는 것들 가볍게 관리하기 (context rot · 200줄 룰)
6. 인건비(토큰) 가볍게 쓰기 (clear · compact · rewind)
7. 부록 (치트시트 · 한 주 적용 플랜)
